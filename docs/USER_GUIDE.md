# User Guide

This guide is under development. Please refer to the installation guide and API documentation.

## Overview

PwnHub provides a web-based interface for managing multiple Pwnagotchi devices.

## Features

- Device registration and management
- Handshake file storage and download
- Web-based SSH access (coming soon)
- Automated backup and restore (coming soon)

## Using the Web Interface

Access the web interface at `http://localhost:8080` after starting the services.
